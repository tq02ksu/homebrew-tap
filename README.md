# tq02ksu Tap

## How do I install these formulae?

```bash
brew tap tq02ksu/tap
brew install tq02ksu/tap/hugrs
```

This tap currently provides:

- `hugrs`

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tq02ksu/tap"
brew "tq02ksu/tap/hugrs"
```
